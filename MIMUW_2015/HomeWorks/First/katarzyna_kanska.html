2. Wyznacz obliczeniowo przedział ufności dla współczynnika nachylenia.

Posłużę się metodą boostrapu resztowego.

set.seed(7)
N <- 5000
nwsp <- replicate(N, {
  ndf <- data.frame(x = heights$Wife,
                    y = model$fitted.values + sample(model$residuals))
  model <- lm(y~x, data=ndf)
  model$coefficients
})

df <- data.frame(slope = nwsp[2,])
hist(df$slope, 100, col="grey")


quantile(df$slope, probs=c(0.025, 0.975))

##      2.5%     97.5% 
## 0.6918457 0.9780119

5% przedział ufności dla współczynnika nachylenia to [0,6918457 ; 0,9780119].
